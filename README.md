# Sistema de Inventario Personal

Un programa para llevar un registro de los artículos que posees.

## Funcionalidades

*   Añadir artículos con descripción, cantidad, precio de compra y fecha de compra.
*   Listar artículos (por categoría, fecha de compra, precio, etc.).
*   Modificar artículos.
*   Eliminar artículos.
*   Buscar artículos.

## Instalación

1.  Clona el repositorio: `git clone <URL_DEL_REPOSITORIO>`
2.  Asegúrate de tener Python instalado.

## Uso

Ejecuta el programa: 

`python inventario.py`
